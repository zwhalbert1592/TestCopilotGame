# Assets Folder

This folder contains all game assets organized by type.

## Structure

- **Sprites/** - Sprite sheets and individual sprite images
- **Audio/** - Sound effects and music files
- **Data/** - Game data files (JSON, XML, etc.)
- **Fonts/** - Font files for text rendering

## Guidelines

- Keep sprite sheets organized by category (characters, enemies, tiles, etc.)
- Use consistent naming conventions
- Document sprite sheet layouts in accompanying .txt or .json files
